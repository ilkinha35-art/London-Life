# London Life 🎮

Um jogo estilo arcade feito em Python usando a biblioteca [arcade](https://api.arcade.academy/).  
O objetivo é atravessar ruas movimentadas de Londres, desviando de carros, ônibus e caminhões que fluem pelo cenário.
