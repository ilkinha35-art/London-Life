London Life 🎮

Um jogo arcade em Python com carros, ônibus e caminhões fluindo

Sobre o projeto

London Life é um protótipo de jogo estilo arcade, onde o jogador controla um personagem que precisa atravessar ruas movimentadas com carros, ônibus e caminhões.

Como rodar

1. Instale Python 3.10+

2. Instale a biblioteca arcade:

pip install arcade

3. Execute:

python src/main.py

Estrutura do projeto

  • src/ → código principal
  • docs/ → documentação
  • README.md → explicação do projeto
  • index.html → página inicial
